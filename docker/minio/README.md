# MinIO Docker Configuration

MinIO S3-compatible object storage configuration.

## Files

- `policies/` - Bucket policies (optional)

## Default Buckets

Created automatically by the application:
- `media-bot-incoming` - Incoming media
- `media-bot-processed` - Processed output
- `media-bot-samples` - Validation samples

## Access

Web UI: http://localhost:9001
API: http://localhost:9000
